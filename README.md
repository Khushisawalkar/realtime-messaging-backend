# Realtime Messaging & Presence Backend

A backend system for real-time messaging built using **Spring Boot, WebSocket, and MySQL**.  
The application enables instant message communication, persistent chat storage, and REST APIs for chat management.

---

## Tech Stack

- Java 17
- Spring Boot
- Spring Web
- Spring WebSocket
- Spring Data JPA
- MySQL
- Maven

---

## Features

- Real-time messaging using WebSocket
- REST APIs for sending and retrieving messages
- Persistent chat storage using MySQL
- Clean layered architecture (Controller → Service → Repository)
